PBSViewer

Version 1.2.2.2

Punkbuster screenshot viewer

Preface

Welcome, thank you very much for downloading this tool. I have put a lot of hours into this project. This tool is based on an older program which I had used for my clan. Now I have made a totally new version with better updating features and less bugs. I hope you will enjoy it, so let's start!

Installation

  1. Unzip package to a new directory.
  2. Create a database.
  3. Upload everything.
  4. CHMOD directory 'inc' to 777.
  5. CHMOD file 'lastUpdate.txt' to 666.
  6. CHMOD directory 'download' to 777.
  7. Please check if you followed all the above mentioned steps correctly. If so then open 'install\install.php' in your browser and follow the instructions. If you don't succeed to install succesfully then go to contact section.
  8. Delete 'install' directory.
  9. CHMOD directory 'inc' to 755.
  10. If you run the script for the first time, you need to update manually. But before you update do the following thing first if needed*.
  11. Open 'reset.php' and click on reset.
  12. Now you can go update manually. Do this after you have played in your gameserver for a while. Just open 'index.php' in your browser and click on update.

*It is recommended to make a back-up of your files located in 'pb/svss' and 'pb/svlogs' located on your gameserver. It is also recommended to delete those screens and log files before updating. Especially if you have more than 100 screens. Otherwise it will take you a long time to download all the screens and log files. Those screens and logs are deleted in step 11 with use of 'reset.php'.

Updating

It is recommended to do a reinstall instead of updating, because config might have changed.
Please before updating make a backup of files you have modified.

Updating from version 1.2.2.1 to 1.2.2.2

  1. Upload the new PBSViewer files.
  2. CHMOD file 'lastUpdate.txt' (if needed) to 666.
  3. CHMOD directory 'download' (if needed) to 777.

System recommendations

  • php version 5.0 or higher.
  • MySQL version 4.0 or higher.
  • Webhosting which supports .htaccess files.
  • Punkbuster server with ftp acces to your pb screens directory.
