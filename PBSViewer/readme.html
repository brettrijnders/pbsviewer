Installation

Follow the installation steps below carefully. If something is not clear please first check the FAQ page to find your answer. If you can not find your answer please go to the contact section

  1. Unzip package to a new directory.
  2. Create a database.
  3. Upload everything to your ftp webserver.
  4. CHMOD directory 'inc' to 777.
  5. CHMOD file 'lastUpdate.txt' to 666.
  6. CHMOD directory 'download' to 777.
  7. Please check if you followed all the above mentioned steps correctly. If so then open 'install/install.php' in your browser and follow the instructions. If you don't succeed to install succesfully then go to contact section.
  8. Delete 'install' directory.
  9. CHMOD directory 'inc' back to 755.
  10. If you run the script for the first time, you need to update manually. But before you update do the following thing first if needed*.
  11. Now you can go update manually. Do this after you have played in your gameserver for a while. Just open 'index.php' in your browser and click on update.

*It is recommended to make a back-up of your screens and log files, they are respectively located in 'pb/svss' and 'pb/svlogs' on your ftp gameserver. After back-up it is also recommended to delete those screens and log files before updating. Especially if you have more than 100 screens. Otherwise it will take you a long time to download all the screens and log files. Those screens and logs need to be removed manually, unless you have enabled the reset feature during install. If you have enabled the reset feature then you should be able to delete those logs and screens automatically by using the reset button on the main page.

Updating

If you already have installed version 2.2.0.2 of PBSViewer and you want to update to 2.2.0.3, just do the following:

  1. Upload the files and overwrite everything.
  2. Enjoy your new version.

Contact

In case you have a question/suggestion or found a bug, please leave a comment on my project page or contact me via mail. If something is not clear please first check the FAQ page to find your answer.

System recommendations

  • php version 5.0 or higher.
  • MySQL version 4.0 or higher.
  • Webhosting that supports .htaccess files.
  • Punkbuster server with ftp acces to your pb screens directory.
